Budgetting
Apple Intelligence powered receipt and expense manager

Tech Stack
SwiftUI, SwiftData, Vision OCR, Foundation Models, Charts, PhotosPicker, AppKit/UIKit, async/await

Features
• Scan receipts from Photos or Files
• Extract receipt text using Vision OCR
• Parse receipt details using Apple Foundation Models
• Save expenses with receipt images, OCR text and items
• AI spending insights
• Ask natural language questions about expenses
• Dashboard with charts and budget progress
